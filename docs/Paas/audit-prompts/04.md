## What to look for

### Python services (MAIA, distress-classifier)

- **Auth**: do the classifier endpoints require an auth token? The DWA
  app calls them internally — verify the service only listens on the
  internal Swarm network, not publicly exposed.
- **PHI in logs**: the classifier receives user text. Verify logging is
  off by default for text content.
- **Deserialization risks**: `pickle`, `yaml.load` (unsafe), `eval` on
  request data.
- **Injection**: SQL / shell / OS command built from request data.
- **Timeout / OOM**: model inference with no timeout = DoS vector.
- **Dockerfile**: runs as root? copies secrets? uses `:latest` tags?
- **Health endpoints**: leak version / config / model-name info?

### Forum (Flarum)

- PHP file permissions, included files, any `exec`/`system`/`eval`.
- JS config files — hardcoded admin tokens?
- Webhook receivers — signature verification?

### Admin scripts

- `db-migrate.ts` — connection string handling, authenticated?
- `generate-openapi.ts` — reveals internal routes?
- `snapshot-platform.ts` — collects what data?
- `validate-curriculum.ts`, `generate_curriculum.py` — input trusted?
- Any script that writes to prod by default.

### `instrumentation.ts`

Already read in Phase 1 — re-audit for keep-alive / self-ping loops
that could hammer a public endpoint + leak env-presence via log format.

### `proxy.ts` (was `middleware.ts`)

Already fixed in Phase 1 for tenant-header spoofing — re-verify the
strip is intact and no new header-trusting code was added. Matcher
should not exclude `/api/*`.

### `next.config.js`

- CSP inline. Re-verify `script-src`, `connect-src`, `frame-src`.
- `'unsafe-inline'` acceptable only in dev.

### e2e Playwright tests

- Tests hitting live production by default? Any `baseURL` pointing at
  prod?
- Hardcoded test credentials?
- Tests that bypass auth (via session cookie mocking) must only run
  when `NEXT_PUBLIC_MOCK_AUTH=true` AND `NODE_ENV !== 'production'`.

### `public/`

- Any client-side JS with embedded API keys / tokens.
- Any HTML / CSS with admin-only leaks.

### `nixpacks.toml` + `deploy-classifier*.sh`

- Shell-script secrets / unsafe quoting / known-host MITM.

# Prompt 16 — Marketing static sites sweep (~564 files)

Audit of the bundled marketing static sites. These are served publicly
via Next.js static file routing — any XSS or leaked secret here is on
the public internet.

Repo: `/Volumes/ext-data/github/solofame-platform`. Run
`docs/audit-prompts/00.md` first. Follow `CLAUDE.md` audit discipline.

## Scope

- `apps/dwa/websites/` (~400 files across 6 bundled mini-sites:
  tidy-next, digitalwellness-academy, digitalwellness-academy-site,
  simple-next, "GTM OS Site", svtech-site)
- `apps/gtm/website/` (~85 files)
- `apps/gtm/soloframehub-website-feb/` (~79 files)

## Procedure

Most files are HTML / CSS / vendored JS (iconify, alpine,
tailwind-themes). Line-by-line reading is low-signal; instead:

### Step 1 — grep sweep

```bash
# Embedded secrets
grep -rlnE 'sk-[a-zA-Z0-9_-]{20,}|re_[a-zA-Z0-9_-]{20,}|AKIA[A-Z0-9]{16}|Bearer\s+[A-Za-z0-9_.-]{20,}|postgres://|redis://|mongodb://' apps/dwa/websites apps/gtm/website apps/gtm/soloframehub-website-feb

# Inline <script> (could be custom, vs vendored)
grep -rln '<script' apps/dwa/websites apps/gtm/website apps/gtm/soloframehub-website-feb | head -50

# Inline event handlers (XSS candidates when mixed with user-content systems)
grep -rlnE 'on(error|load|click|mouseover|submit)\s*=' apps/dwa/websites apps/gtm/website apps/gtm/soloframehub-website-feb

# Internal URLs / IPs / dokploy
grep -rlnEi 'dokploy|10\.|172\.(1[6-9]|2[0-9]|3[01])\.|192\.168\.|admin\.soloframehub' apps/dwa/websites apps/gtm/website apps/gtm/soloframehub-website-feb

# Analytics / tracking IDs (rotation-policy candidates, not bugs per se)
grep -rlnE 'G-[A-Z0-9]{10,}|GTM-[A-Z0-9]+|fbq\(|gtag\(' apps/dwa/websites apps/gtm/website apps/gtm/soloframehub-website-feb | head -30

# Forms posting to internal endpoints
grep -rlnE '<form[^>]*action=["\x27][^"\x27]+["\x27]' apps/dwa/websites apps/gtm/website apps/gtm/soloframehub-website-feb
```

### Step 2 — drill into every hit

Open each flagged file. Read enough context to classify
real-secret / placeholder / vendored-code-unchanged.

### Step 3 — deep read

Read all top-level `index.html`, `index.tsx`, and any non-template
JS / TS. Skip vendored libraries (iconify/, alpine/, tailwind themes).

### Step 4 — mark the ledger

```bash
find apps/dwa/websites apps/gtm/website apps/gtm/soloframehub-website-feb -type f \
  ! -name '*.png' ! -name '*.jpg' ! -name '*.jpeg' ! -name '*.webp' ! -name '*.svg' \
  ! -name '*.woff*' ! -name '*.ttf' ! -name '*.pdf' ! -name '*.ico' \
  | python3 /tmp/mark-audited.py - clean "static-marketing pattern-swept"
# Then re-mark grep-flagged files as findings.
```

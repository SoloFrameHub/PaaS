## What to look for

Everything in prompt 01, plus GTM-specific:

- **Polar / Stripe / payment paths**: any admin endpoint that grants
  subscriptions, assigns plan tier, or mutates billing state — must be
  admin-gated + audit-logged.
- **Webhook handlers**: signature verification must use
  `timingSafeEqual`. Missing secret must fail-closed (B-029 class —
  the Polar bug).
- **CRM / Attio / Hunter / Pipedrive writes**: these cost money per API
  call (outbound volume). Rate limit + auth gate required.
- **Form submission endpoints**: anti-spam, honeypot, reCAPTCHA?
- **Email sending routes** (`/api/email`, digest, readiness-followup):
  unsubscribe link handling, SPF-friendly sending, from address.
- **Badgr / open-badge issuance**: any endpoint that issues a badge
  must verify the user actually earned it server-side.
- **`/api/pipeline/**` (sales CRM)**: IDOR — one user reading another
  user's deals.
- **Notion OAuth callback**: `state` param verification, CSRF
  protection, PKCE (if applicable), token storage.
- **Silent JSONB dot-path writes** (B-039).

   HIGH finding with the proposed fix and the affected `file:line(s)`.
   Paste the table into the audit doc under `## Remediation plan`.

   pnpm -r --filter "@platform/tenancy" --filter "@platform/manifest-loader" --filter "@platform/prompt-registry" --filter "@platform/testing" test

   `docs/AUDIT-FULL-2026-04-23.md`:

   - all fixes landed this session
   - any finding NOT fixed + why + escalation plan

9. **Print final ledger stats.** Every in-scope file must be `✅` or
   `🔍`. If not, escalate the gap.

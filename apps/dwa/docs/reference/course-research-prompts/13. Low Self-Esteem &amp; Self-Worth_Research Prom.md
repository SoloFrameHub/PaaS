# 13. Low Self-Esteem \& Self-Worth

Research Prompt:
Survey CBT, schema therapy, and compassion-focused therapy literature on low self-esteem, self-criticism, and shame. Identify evidence-based techniques (thought records, behavioral experiments, self-compassion practices, strengths-based reflection) that can be delivered as reusable React components like components/selfEsteem/ThoughtRecordForm.tsx and components/selfEsteem/CompassionLetterWizard.tsx, using strong TypeScript typing for entries and validation. Explore brief, non-identifying self-esteem questionnaires (e.g., items inspired by Rosenberg Self-Esteem Scale without reproducing full copyrighted scales) and design anonymous, on-device scoring logic that returns educational feedback rather than diagnosis. Include Next.js-specific guidance for accessible textarea journaling components, autosave in local storage when appropriate, and messaging that reassures users about anonymity and data privacy

Low self‑esteem, harsh self‑criticism, and shame are maintained by negative core beliefs (“I am unworthy”), rigid self‑standards, and attentional biases to failures; CBT, schema therapy, and compassion‑focused therapy each offer structured, evidence‑based methods to modify these patterns. Compassion‑focused therapy in particular shows meta‑analytic support for reducing self‑criticism and shame and increasing self‑soothing, making it well suited to digital interventions that combine cognitive work with self‑compassion practices.[^1][^2][^3][^4][^5][^6][^7][^8]

## Key therapeutic concepts

- CBT models low self‑esteem as driven by negative core beliefs, conditional assumptions (e.g., “If I fail, it proves I’m useless”), and safety behaviors that prevent disconfirmation.[^4][^9][^1]
- Schema therapy emphasizes the defectiveness/shame schema and associated coping modes (surrender, avoidance, overcompensation) and works to strengthen a Healthy Adult mode that validates needs and challenges toxic self‑beliefs.[^10][^5][^7][^11]
- Compassion‑focused therapy (CFT) targets the “threat system” of shame and self‑attack and cultivates compassionate motivational, affective, and cognitive systems to replace self‑criticism with warmth and support.[^12][^3][^6][^8]

These models consistently support techniques like thought records, behavioral experiments, values‑ and strengths‑based reflection, imagery rescripting, and self‑compassion exercises for improving self‑worth.[^13][^14][^5][^6][^9][^4]

## Evidence‑based techniques to implement

You can safely adapt these techniques without copying any proprietary materials.

- Thought records: Identify triggering situations, automatic self‑critical thoughts, associated emotions and intensities, evidence for/against, balanced alternatives, and action plans.[^15][^13][^4]
- Behavioral experiments: Plan small, testable actions (e.g., “share one idea in a meeting”) to test predictions (“People will think I’m stupid”) against actual outcomes to update beliefs.[^9][^4]
- Self‑compassion practices: CFT‑style exercises cultivate an inner compassionate voice, soothing imagery, and supportive self‑talk, which reduce self‑criticism and shame.[^3][^6][^8]
- Strengths‑based reflection: Guided prompts help users identify past coping, personal strengths, and values‑consistent actions that contradict global “I am a failure” beliefs.[^2][^5][^7]

### ThoughtRecordForm component

```tsx
// components/selfEsteem/ThoughtRecordForm.tsx
'use client';

import React, { useState, useEffect } from 'react';
import type { ThoughtRecordEntry, EmotionLabel, ValidationErrors } from '@/types/selfEsteem';

interface ThoughtRecordFormProps {
  initialValue?: Partial<ThoughtRecordEntry>;
  onSubmit?: (entry: ThoughtRecordEntry) => void;
  enableLocalAutosave?: boolean;
  autosaveKey?: string;
}

const emptyThoughtRecord: ThoughtRecordEntry = {
  id: '',
  createdAt: '',
  situation: '',
  automaticThought: '',
  emotion: 'ashamed',
  emotionIntensity: 50,
  evidenceFor: '',
  evidenceAgainst: '',
  alternativeThought: '',
  actionPlan: '',
};

const emotionOptions: { value: EmotionLabel; label: string }[] = [
  { value: 'sad', label: 'Sad' },
  { value: 'anxious', label: 'Anxious / Worried' },
  { value: 'ashamed', label: 'Ashamed / Embarrassed' },
  { value: 'angry', label: 'Angry' },
  { value: 'disgusted', label: 'Disgusted' },
  { value: 'guilty', label: 'Guilty' },
  { value: 'other', label: 'Other' },
];

function validate(entry: ThoughtRecordEntry): ValidationErrors<ThoughtRecordEntry> {
  const errors: ValidationErrors<ThoughtRecordEntry> = {};
  if (!entry.situation.trim()) errors.situation = 'Please briefly describe the situation.';
  if (!entry.automaticThought.trim()) errors.automaticThought = 'Please enter the main self-critical thought.';
  if (entry.emotionIntensity < 0 || entry.emotionIntensity > 100) {
    errors.emotionIntensity = 'Please choose a value between 0 and 100.';
  }
  if (!entry.alternativeThought.trim()) errors.alternativeThought = 'Try writing at least one kinder, more balanced thought.';
  return errors;
}

export const ThoughtRecordForm: React.FC<ThoughtRecordFormProps> = ({
  initialValue,
  onSubmit,
  enableLocalAutosave = true,
  autosaveKey = 'rps-selfEsteem-thoughtRecordDraft',
}) => {
  const [entry, setEntry] = useState<ThoughtRecordEntry>({
    ...emptyThoughtRecord,
    ...initialValue,
  });
  const [errors, setErrors] = useState<ValidationErrors<ThoughtRecordEntry>>({});
  const [statusMessage, setStatusMessage] = useState<string | null>(null);

  // Load draft from localStorage (client-side only)
  useEffect(() => {
    if (!enableLocalAutosave) return;
    if (typeof window === 'undefined') return;
    const raw = window.localStorage.getItem(autosaveKey);
    if (raw) {
      try {
        const parsed = JSON.parse(raw) as ThoughtRecordEntry;
        setEntry((prev) => ({ ...prev, ...parsed }));
      } catch {
        // ignore parse errors
      }
    }
  }, [enableLocalAutosave, autosaveKey]);

  // Autosave
  useEffect(() => {
    if (!enableLocalAutosave) return;
    if (typeof window === 'undefined') return;
    const timeout = window.setTimeout(() => {
      window.localStorage.setItem(autosaveKey, JSON.stringify(entry));
      setStatusMessage('Draft saved privately on this device.');
    }, 800);
    return () => window.clearTimeout(timeout);
  }, [entry, enableLocalAutosave, autosaveKey]);

  function handleChange<K extends keyof ThoughtRecordEntry>(key: K, value: ThoughtRecordEntry[K]) {
    setEntry((prev) => ({ ...prev, [key]: value }));
  }

  function handleSubmit(e: React.FormEvent) {
    e.preventDefault();
    const withMeta: ThoughtRecordEntry = {
      ...entry,
      id: entry.id || crypto.randomUUID(),
      createdAt: entry.createdAt || new Date().toISOString(),
    };
    const validation = validate(withMeta);
    setErrors(validation);
    if (Object.keys(validation).length > 0) {
      setStatusMessage('Please review the highlighted fields.');
      return;
    }
    onSubmit?.(withMeta);
    setStatusMessage('Thought record completed. Notice any shifts in emotion or perspective.');
  }

  return (
    <form onSubmit={handleSubmit} aria-describedby="tr-privacy tr-purpose">
      <p id="tr-purpose">
        Use this private exercise to gently question a self-critical thought and practice a more balanced view.
      </p>
      <p id="tr-privacy">
        Your responses stay on your device unless you choose to export or share them. Avoid entering names, locations, or other identifying details.
      </p>

      <label>
        Situation (what was happening?)
        <textarea
          value={entry.situation}
          onChange={(e) => handleChange('situation', e.target.value)}
          rows={3}
        />
        {errors.situation && <span role="alert">{errors.situation}</span>}
      </label>

      <label>
        Main self-critical thought
        <textarea
          value={entry.automaticThought}
          onChange={(e) => handleChange('automaticThought', e.target.value)}
          rows={3}
        />
        {errors.automaticThought && <span role="alert">{errors.automaticThought}</span>}
      </label>

      <label>
        Main emotion
        <select
          value={entry.emotion}
          onChange={(e) => handleChange('emotion', e.target.value as EmotionLabel)}
        >
          {emotionOptions.map((opt) => (
            <option key={opt.value} value={opt.value}>
              {opt.label}
            </option>
          ))}
        </select>
      </label>

      <label>
        Emotion intensity (0–100)
        <input
          type="range"
          min={0}
          max={100}
          value={entry.emotionIntensity}
          onChange={(e) => handleChange('emotionIntensity', Number(e.target.value))}
        />
        <span aria-live="polite">{entry.emotionIntensity}</span>
        {errors.emotionIntensity && <span role="alert">{errors.emotionIntensity}</span>}
      </label>

      <label>
        Evidence that seems to support the thought
        <textarea
          value={entry.evidenceFor}
          onChange={(e) => handleChange('evidenceFor', e.target.value)}
          rows={3}
        />
      </label>

      <label>
        Evidence that does not fit the thought
        <textarea
          value={entry.evidenceAgainst}
          onChange={(e) => handleChange('evidenceAgainst', e.target.value)}
          rows={3}
        />
      </label>

      <label>
        A kinder, more balanced alternative thought
        <textarea
          value={entry.alternativeThought}
          onChange={(e) => handleChange('alternativeThought', e.target.value)}
          rows={3}
        />
        {errors.alternativeThought && <span role="alert">{errors.alternativeThought}</span>}
      </label>

      <label>
        Small action you can take that fits this new perspective
        <textarea
          value={entry.actionPlan}
          onChange={(e) => handleChange('actionPlan', e.target.value)}
          rows={2}
        />
      </label>

      <button type="submit">Save Thought Record</button>

      {statusMessage && <p aria-live="polite">{statusMessage}</p>}
    </form>
  );
};
```

This component integrates CBT thought records, emotion labeling, and privacy‑sensitive, local‑only autosave, while returning psychoeducational feedback instead of any diagnosis.[^13][^4][^15][^9]

### CompassionLetterWizard component

```tsx
// components/selfEsteem/CompassionLetterWizard.tsx
'use client';

import React, { useState, useEffect } from 'react';
import type { CompassionLetterDraft, ValidationErrors } from '@/types/selfEsteem';

interface CompassionLetterWizardProps {
  initialValue?: Partial<CompassionLetterDraft>;
  onComplete?: (draft: CompassionLetterDraft) => void;
  autosaveKey?: string;
}

type Step = 1 | 2 | 3 | 4;

const emptyDraft: CompassionLetterDraft = {
  id: '',
  createdAt: '',
  promptVersion: 'v1',
  selfDescription: '',
  compassionateVoice: '',
  encouragements: '',
  commitments: '',
};

function validateDraft(draft: CompassionLetterDraft): ValidationErrors<CompassionLetterDraft> {
  const errors: ValidationErrors<CompassionLetterDraft> = {};
  if (!draft.selfDescription.trim()) errors.selfDescription = 'Describe briefly how you’ve been feeling about yourself.';
  if (!draft.compassionateVoice.trim()) errors.compassionateVoice = 'Try writing at least two sentences from a kind, understanding perspective.';
  return errors;
}

export const CompassionLetterWizard: React.FC<CompassionLetterWizardProps> = ({
  initialValue,
  onComplete,
  autosaveKey = 'rps-selfEsteem-compassionLetterDraft',
}) => {
  const [draft, setDraft] = useState<CompassionLetterDraft>({ ...emptyDraft, ...initialValue });
  const [step, setStep] = useState<Step>(1);
  const [errors, setErrors] = useState<ValidationErrors<CompassionLetterDraft>>({});
  const [status, setStatus] = useState<string | null>(null);

  useEffect(() => {
    if (typeof window === 'undefined') return;
    const raw = window.localStorage.getItem(autosaveKey);
    if (raw) {
      try {
        const parsed = JSON.parse(raw) as CompassionLetterDraft;
        setDraft((prev) => ({ ...prev, ...parsed }));
      } catch {
        // ignore
      }
    }
  }, [autosaveKey]);

  useEffect(() => {
    if (typeof window === 'undefined') return;
    const timeout = window.setTimeout(() => {
      window.localStorage.setItem(autosaveKey, JSON.stringify(draft));
      setStatus('Draft saved locally. You can close this page and return later on this device.');
    }, 800);
    return () => window.clearTimeout(timeout);
  }, [draft, autosaveKey]);

  function update<K extends keyof CompassionLetterDraft>(key: K, value: CompassionLetterDraft[K]) {
    setDraft((prev) => ({ ...prev, [key]: value }));
  }

  function goNext() {
    setStep((prev) => (Math.min(prev + 1, 4) as Step));
  }

  function goPrev() {
    setStep((prev) => (Math.max(prev - 1, 1) as Step));
  }

  function handleFinish() {
    const withMeta: CompassionLetterDraft = {
      ...draft,
      id: draft.id || crypto.randomUUID(),
      createdAt: draft.createdAt || new Date().toISOString(),
    };
    const validation = validateDraft(withMeta);
    setErrors(validation);
    if (Object.keys(validation).length > 0) {
      setStatus('Please complete the required steps before finishing.');
      return;
    }
    onComplete?.(withMeta);
    setStatus('Compassion letter completed. Many people find it helpful to reread this when self-criticism shows up.');
  }

  return (
    <section aria-labelledby="clw-heading" aria-describedby="clw-privacy">
      <h2 id="clw-heading">Self-Compassion Letter</h2>
      <p id="clw-privacy">
        This gentle exercise comes from compassion-focused therapy. Your writing is stored only in your browser unless you export it. Please avoid including names, addresses, or other identifying details.
      </p>

      <nav aria-label="Steps">
        <ol>
          <li aria-current={step === 1 ? 'step' : undefined}>1. Notice self-criticism</li>
          <li aria-current={step === 2 ? 'step' : undefined}>2. Imagine a compassionate friend</li>
          <li aria-current={step === 3 ? 'step' : undefined}>3. Write encouragement</li>
          <li aria-current={step === 4 ? 'step' : undefined}>4. Commit to small acts of care</li>
        </ol>
      </nav>

      {step === 1 && (
        <div>
          <h3>Step 1: How have you been speaking to yourself?</h3>
          <p>Briefly describe the situation and the kind of things your inner critic has been saying.</p>
          <label>
            Your description
            <textarea
              rows={5}
              value={draft.selfDescription}
              onChange={(e) => update('selfDescription', e.target.value)}
            />
            {errors.selfDescription && <span role="alert">{errors.selfDescription}</span>}
          </label>
        </div>
      )}

      {step === 2 && (
        <div>
          <h3>Step 2: A compassionate perspective</h3>
          <p>
            Imagine someone who truly cares about you and sees your struggles. What would they say to you about this situation,
            knowing your history and limitations?
          </p>
          <label>
            Compassionate voice
            <textarea
              rows={5}
              value={draft.compassionateVoice}
              onChange={(e) => update('compassionateVoice', e.target.value)}
            />
            {errors.compassionateVoice && <span role="alert">{errors.compassionateVoice}</span>}
          </label>
        </div>
      )}

      {step === 3 && (
        <div>
          <h3>Step 3: Encouragement and understanding</h3>
          <p>Write a few sentences acknowledging your efforts and normalizing your struggles as part of being human.</p>
          <label>
            Encouraging messages
            <textarea
              rows={4}
              value={draft.encouragements}
              onChange={(e) => update('encouragements', e.target.value)}
            />
          </label>
        </div>
      )}

      {step === 4 && (
        <div>
          <h3>Step 4: Small acts of care</h3>
          <p>List one or two small, realistic actions you can take to support yourself over the next few days.</p>
          <label>
            Commitments to yourself
            <textarea
              rows={4}
              value={draft.commitments}
              onChange={(e) => update('commitments', e.target.value)}
            />
          </label>
        </div>
      )}

      <div>
        {step > 1 && <button type="button" onClick={goPrev}>Back</button>}
        {step < 4 && <button type="button" onClick={goNext}>Next</button>}
        {step === 4 && (
          <button type="button" onClick={handleFinish}>
            Finish Letter
          </button>
        )}
      </div>

      {status && <p aria-live="polite">{status}</p>}
    </section>
  );
};
```

This wizard reflects CFT principles (noticing self‑criticism, activating a compassionate other, offering encouragement, planning self‑care) and the evidence that CFT reduces self‑criticism and shame and increases self‑soothing.[^6][^8][^3]

## Brief, anonymous self‑esteem questionnaire

To stay clear of copyright, you can design a 6–8 item check‑in inspired by common self‑esteem constructs (global worth, self‑respect, self‑acceptance) without reproducing any specific scale.[^1][^9]

Example items (Likert 0–4: “Not at all true” to “Very often true”):

- “I speak to myself in a harsh or insulting way.”
- “When I make a mistake, I treat it as proof that I am a failure.”
- “I find it hard to notice my strengths or positive qualities.”
- “I feel basically worthy of kindness and respect.” (reverse‑scored)

Client‑side scoring logic can simply sum items and map ranges to educational feedback such as “Right now you may be very hard on yourself; many people in this range find self‑compassion and thought‑challenging exercises helpful” without giving any diagnosis.[^3][^6][^9][^1]

### Sketch of on‑device questionnaire component

```ts
// types/selfEsteemQuestionnaire.ts
export interface SelfEsteemItem {
  id: string;
  prompt: string;
  reverseScored?: boolean;
}

export interface SelfEsteemResponse {
  itemId: string;
  value: 0 | 1 | 2 | 3 | 4;
}

export interface SelfEsteemResult {
  total: number;
  max: number;
  category: 'low' | 'moderate' | 'higher';
  message: string;
}
```

```tsx
// components/selfEsteem/SelfEsteemCheckIn.tsx
'use client';

import React, { useState } from 'react';
import type { SelfEsteemItem, SelfEsteemResponse, SelfEsteemResult } from '@/types/selfEsteemQuestionnaire';

const items: SelfEsteemItem[] = [
  { id: 'harsh', prompt: 'I speak to myself in a harsh or insulting way.' },
  { id: 'failure', prompt: 'When I make a mistake, I treat it as proof that I am a failure.' },
  { id: 'strengths', prompt: 'I find it hard to notice my strengths or positive qualities.' },
  { id: 'respect', prompt: 'I feel basically worthy of kindness and respect.', reverseScored: true },
  { id: 'compare', prompt: 'I often compare myself unfavorably to other people.' },
  { id: 'hopeful', prompt: 'I can imagine a future where I feel more accepting of myself.', reverseScored: true },
];

const choices: { value: 0 | 1 | 2 | 3 | 4; label: string }[] = [
  { value: 0, label: 'Not at all true' },
  { value: 1, label: 'Rarely true' },
  { value: 2, label: 'Sometimes true' },
  { value: 3, label: 'Often true' },
  { value: 4, label: 'Very often true' },
];

function score(responses: SelfEsteemResponse[]): SelfEsteemResult | null {
  if (responses.length !== items.length) return null;
  let raw = 0;
  for (const item of items) {
    const r = responses.find((x) => x.itemId === item.id);
    if (!r) return null;
    const v = item.reverseScored ? 4 - r.value : r.value;
    raw += v;
  }
  const max = items.length * 4;
  const percentage = raw / max;

  let category: SelfEsteemResult['category'];
  let message: string;

  if (percentage <= 0.33) {
    category = 'low';
    message =
      'These answers suggest you may be very hard on yourself right now. This check-in is not a diagnosis, but it can be a sign that gentle support, self-compassion work, and professional help could be useful.';
  } else if (percentage <= 0.66) {
    category = 'moderate';
    message =
      'Your responses suggest a mix of self-doubt and self-acceptance. It may help to practice noticing strengths and gently questioning harsh self-judgments.';
  } else {
    category = 'higher';
    message =
      'Your responses suggest a relatively kinder attitude toward yourself, even if doubts still show up at times. Maintaining supportive self-talk and compassion practices can help you stay steady.';
  }

  return { total: raw, max, category, message };
}

export const SelfEsteemCheckIn: React.FC = () => {
  const [responses, setResponses] = useState<SelfEsteemResponse[]>([]);
  const [result, setResult] = useState<SelfEsteemResult | null>(null);

  function setResponse(itemId: string, value: 0 | 1 | 2 | 3 | 4) {
    setResponses((prev) => {
      const others = prev.filter((r) => r.itemId !== itemId);
      return [...others, { itemId, value }];
    });
  }

  function handleSubmit(e: React.FormEvent) {
    e.preventDefault();
    const scored = score(responses);
    if (scored) setResult(scored);
  }

  return (
    <section aria-labelledby="se-heading" aria-describedby="se-desc se-privacy">
      <h2 id="se-heading">Self-Esteem Check-In</h2>
      <p id="se-desc">
        This brief check-in explores how you relate to yourself. It gives educational feedback only and is not a diagnosis or a substitute for professional care.
      </p>
      <p id="se-privacy">
        Your answers are processed only on this device and are not sent to a server. Please avoid entering names or other identifying information.
      </p>

      <form onSubmit={handleSubmit}>
        {items.map((item) => (
          <fieldset key={item.id}>
            <legend>{item.prompt}</legend>
            {choices.map((choice) => (
              <label key={choice.value}>
                <input
                  type="radio"
                  name={item.id}
                  value={choice.value}
                  onChange={() => setResponse(item.id, choice.value)}
                />
                {choice.label}
              </label>
            ))}
          </fieldset>
        ))}

        <button type="submit">Show Feedback</button>
      </form>

      {result && (
        <section aria-live="polite">
          <h3>What this score means</h3>
          <p>{result.message}</p>
          <p>
            Total score: {result.total} out of {result.max}. Remember, this is only a guide to help you reflect, not a label or diagnosis.
          </p>
        </section>
      )}
    </section>
  );
};
```

This pattern respects copyright by avoiding any specific proprietary item set, uses on‑device scoring, and returns psychoeducational guidance only.[^6][^9][^1][^3]

## Next.js accessibility, journaling, and privacy messaging

For Next.js specifically:

```
- Use semantic elements (`<form>`, `<label>`, `<fieldset>`, `<legend>`, `<textarea>`) and connect labels with `htmlFor`/`id` to support screen readers.[^12][^4]
```

- Wrap dynamic feedback in elements with `aria-live="polite"` for non‑disruptive announcements (e.g., autosave messages, score explanations).
- For autosave in `localStorage`, guard on `typeof window !== 'undefined'` to avoid hydration mismatches in Next.js, and clearly state that drafts are stored only in the browser and can be cleared any time.
- Provide short privacy banners on all components: emphasize that entries are anonymous, advise avoiding names/identifiers, and clarify that content is not monitored in real time and does not replace crisis support.

By combining these elements, you can deliver CBT, schema‑informed, and compassion‑focused tools for low self‑esteem that are clinically grounded, anonymous by design, and technically well‑integrated into a modern Next.js‑based Digital Wellness platform.[^5][^7][^2][^4][^9][^1][^3][^6]

# Route Migration Map

This document maps the old Express-style URLs to the new Next.js App Router structure.

## Community Pages
| Feature | Old URL | New URL | File Path |
| :--- | :--- | :--- | :--- |
| Feed | `/community` | `/community/feed` | `app/(default)/community/feed/page.tsx` |
| Feed (Direct) | `/community/feed` | `/community/feed` | `app/(default)/community/feed/page.tsx` |
| Forum | `/community/forum` | `/community/forum` | `app/(default)/community/forum/page.tsx` |
| Meetups | `/community/meetups` | `/community/meetups` | `app/(default)/community/meetups/page.tsx` |
| Profile | `/profile` | `/community/profile` | `app/(double-sidebar)/community/profile/page.tsx` |

> [!NOTE]
> `/community` now returns a 404 because it is only a route group folder. All community tests must be updated to use `/community/feed` or `/community/forum` directly.
